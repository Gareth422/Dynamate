DYNAMATE HISTORY
A couple of years ago me and my friend Björn Kalzen developed a game for the Amiga. Björn programmed and made levels, while I worked on creating the graphics, music, and levels.

The game was released as licenceware by F1 in the U.K. but only eleven people discovered the game. That's not a developers dream -eleven users :-). We talked about putting it on the Aminet, but none of us had any internet experience nor connections, so we never did.

However, the game never really left my thoughts so when I registered as a Be developer I needed an easy project to start with; this game was the obvious choice.
